# Reject Rejection

A browser-based tool that helps job seekers turn generic rejection emails into witty, risk-scored reply drafts — without sending anything automatically.

## Live Demo
https://reject-rejection.netlify.app/

## Why I built this
Job seekers receive many generic rejection emails after interviews, assignments, or ghosting. This tool gives candidates a small moment of control, humor, and emotional relief while keeping communication safe and reviewable.

## Key Features
- Paste a rejection email or enter quick context manually
- Choose the situation: just applied, interviewed, assignment, ghosted, no idea
- Generate different tones: professional, savage, LinkedIn-style, etc.
- Risk-aware reply drafting
- Local-first privacy: nothing is uploaded or logged
- Rage Archive for saving drafts locally

## Product Thinking
This is not designed to automatically attack HR. The product principle is:
"Give users emotional release, but keep them in control."

## Tech Stack
- Frontend: HTML / CSS / JavaScript or React
- Deployment: Netlify
- Design: responsive single-page app
